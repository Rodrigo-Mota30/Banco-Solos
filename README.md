# Banco-Solos
Banco Solos 🌱 Fintech sustentável desenvolvida com HTML5, CSS3 e Tailwind CSS. O projeto traz uma arquitetura SPA (Single Page Application) fluida com navegação dinâmica entre Home, Extrato com faturas a pagar, Gestão de Cartões, Investimentos e telas de Acesso. Código limpo, modular e responsivo.
